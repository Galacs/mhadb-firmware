# MHADB Firmware

This repo contains code for the MHADB line following robot, there is the firmware for the line sensor, the FOC bldc driver and the main board.

## Features

- CAN communication between all mcus using common library
- PID based line following algorithm
- Wifi serial monitoring using ESP-NOW
- BLDC Foc control using SimpleFoc
- Buzzer music playback from converted midi files
- LED effects using FastLED
